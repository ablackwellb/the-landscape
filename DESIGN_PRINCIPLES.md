# Design Principles
## This document explains what The Landscape refuses to do, and why. It exists because the most important decisions in this tool are the ones you can't see in the features that were designed out.

> The one rule: Reflect, never generate. The tool renders what a person already knows about their own life. It never adds an interpretation, conclusion, or judgment of its own. Every principle below is this rule applied to a specific temptation.

## What the tool refuses to do:

**It never sets a depth.** Every valley and peak is placed by the person's own hand. No algorithm decides how much something hurt or how much it mattered. 

**It never scores.** There is no resilience metric, no growth index, no resonance number. A score would be the tool claiming to know something about the person that they didn't tell it, and delivering that claim with the false authority of mathematics.

**It never interprets.** The tool does not connect a person's dots for them. There are no detected patterns, no "your hard seasons cluster around X", and no causal stories. A pattern the person finds themselves is insight. The same pattern announced by a machine is a verdict, and this tool issues no verdicts.

**It never predicts.** No trend lines, no trajectory claims, no promises that the path rises. Some honest landscapes don't trend upward, and the person whose doesn't has not failed. and must never be shown a tool that implies they have.

**It never diagnoses, treats, or advises.** This is not therapy and does not perform therapy's moves. It offers no coping strategies, no reframes, no guidance. If a person needs support, they need a person and nothing in this tool should ever feel like a substitute for one.

**It never keeps what it's shown.** Nothing entered is transmitted, stored, or seen by anyone but the person. The only copy that can exist is the one they deliberately save to their own device. This is verifiable: the entire tool is one readable file with no server behind it.

## Why the restraint:

A tool this close to a person's pain has exactly one honest posture: a mirror, not an oracle. Every rejected feature above was rejected because it would have made the tool the author of the person's story instead of a surface that reflects it. The belief that hard seasons can deepen a life may be true, but it is a belief a person is free to find in their own terrain, never a conclusion the terrain is engineered to hand them.

The measure of this design is not what it can compute. It is that a person can stand in front of their whole life. The valleys at true depth and the path is unbroken. Everything they will see there is theirs and theirs alone.
